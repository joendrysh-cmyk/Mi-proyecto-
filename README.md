# Instalación de ani-cli

Para instalar este cliente de anime en sistemas basados en Unix (Linux/Arch), sigue estos pasos:

### 1. Instalar dependencias
Asegúrate de tener instalados `git`, `grep`, `sed`, `curl` y un reproductor como `mpv`.

### 2. Comandos de instalación:
```bash
git clone "https://github.com/pystardust/ani-cli.git"
sudo cp ani-cli/ani-cli /usr/local/bin
rm -rf ani-cli
